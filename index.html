Hexo+github+Hexo editor搭建自己的博客（Mac）

Posted on 2018-10-05

#分享是提高认知的唯一途径#

相信很多技术咖一直都有很多idea，但总会选择拖延式实现，慢慢的，最后选择让这些idea消磨殆尽。有人说，灵感需要捕抓，有思想的人或者更充实吧。说干就干，我今天给大家分享一下自己搭建Hexo的全过程。

1.用Hexo在本地搭建一个博客

1.1 下载并安装nodejs

1.2 安装git

1.3 打开终端使用npm安装hexo

  • sudo -i //切换root权限
  • npm install -g hexo-cli
  • //等待success

快速开始

在终端输入

  • hexo i blog //i是init的缩写，blog是你的博客项目名，你可以自己自定义
  • cd blog //切换到站点根目录
  • hexo g //g是generetor的缩写，生成默认静态文件
  • hexo s //s是server的缩写，部署代码到本地

在浏览器打开输入 localhost:4000 即可查看

2.Hexo主题选择——next

2.1 在站点根目录输入：

  • git clone https://github.com/iissnan/hexo-theme-next themes/next

2.2 完成后，打开站点的配置文件，走到theme字段，并将其值改为 next

2.3 在终端输入

  • hexo clean //清除缓存
  • hexo g //重新生成代码
  • hexo s //部署到本地

//在浏览器打开输入 localhost:4000 查看效果

next主题有三种，你可自己选择切换使用：

  • Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
  • Mist - Muse 的紧凑版本，整洁有序的单栏外观
  • Pisces - 双栏 Scheme，小家碧玉似的清新
